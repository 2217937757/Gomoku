# 五子棋游戏 - 人机对战

## 📖 项目简介

这是一款**人机对战五子棋游戏**，采用 Python + Pygame 开发。具有智能 AI、实时计时、自动保存、棋谱管理等特色功能。

![Python](https://img.shields.io/badge/Python-3.10+-green)
![Pygame](https://img.shields.io/badge/Pygame-2.6.1-red)

## ✨ 核心功能

### 🎮 游戏功能
- ✅ **人机对战** - 玩家 vs AI 智能对手

## 📁 文件说明

### 生成的文件格式

#### 1. **自动保存的棋谱**
```
gomoku_YYYYMMDD_HHMMSS.txt
```
- **触发时机**: 每局游戏结束时自动保存
- **命名规则**: `gomoku_年月日_时分秒.txt`
- **示例**: `gomoku_20260402_121134.txt`

#### 2. **手动保存的棋谱**
```
gomoku_YYYYMMDD_HHMMSS.txt
```
- **触发时机**: 点击菜单"保存棋谱"时
- **命名规则**: 与自动保存相同
- **提示信息**: 弹窗显示"✓ 保存成功！"

### 棋谱文件格式

```txt
╔══════════════════════════════════════╗
║          五子棋棋谱                  ║
╚══════════════════════════════════════╝

对局时间：2026-04-02 12:11:34
总步数：14
结果：AI 获胜
玩家用时：0:02
AI 用时：0:00
难度：困难
保存时间：2026-04-02 12:11:34

══════════════════════════════════════════════════

第  1 步 ● 玩家   -> F10  (用时：0.67 秒)
第  2 步 ○ AI   -> E10  (用时：0.03 秒)
第  3 步 ● 玩家   -> H10  (用时：0.27 秒)
...
```

## 🧠 AI 算法说明

### 评估机制

AI 使用**基于规则的评分系统**来评估每个候选位置：

#### 评分权重
```python
FIVE (连五):     1,000,000 分  ← 必胜
FOUR (活四):       100,000 分  ← 极高威胁
THREE (活三):       10,000 分  ← 进攻机会
TWO (活二):          1,000 分  ← 布局阶段
BLOCK_FOUR (冲四):   90,000 分  ← 防守重点
BLOCK_THREE(眠三):   9,000 分  ← 需要阻挡
```

### 难度设置

| 难度 | 搜索范围 | 反应时间 | 特点 |
|------|----------|----------|------|
| **简单** | 周围 1 格 | ~0.3 秒 | 适合新手 |
| **困难** | 周围 2 格 | ~1.5 秒 | 更具挑战性 |

## 🛠️ 高级功能

### 1. 自动保存机制

**触发条件：**
- ✅ 玩家获胜时
- ✅ AI 获胜时
- ✅ 平局时

**特点：**
- 静默保存，不弹窗打扰
- 控制台输出：`[自动保存] 对局结束 -> gomoku_*.txt`
- 失败时仅记录日志，不影响游戏

### 2. 时间统计系统

**显示内容：**
- **左下角**: 上步用时（玩家或 AI）
- **右下角**: 总步数 + 双方累计用时

**计时精度：**
- 毫秒级记录
- 格式化显示：`M:SS`（分钟：秒）

### 3. 后台 AI 计算

**技术实现：**
- 使用 `threading` 模块创建后台线程
- AI 计算与 UI 渲染分离
- 保持 30 FPS 流畅度

**优势：**
- 界面不卡顿
- 鼠标悬停、菜单操作响应流畅
- 可随时中断（如点击悔棋）

## 🔧 配置说明

### 修改游戏参数

在 `五子棋.py` 文件开头可以调整以下配置：

```python
# 棋盘设置
BOARD_SIZE = 15          # 棋盘大小（15×15）
GRID = 40                # 格子间距（像素）
MARGIN = 40              # 棋盘边距（像素）
MENU_HEIGHT = 28         # 菜单栏高度（像素）

# AI 难度
AI_DIFFICULTY = 2        # 1=简单，2=困难

# 颜色配置
BOARD_BG = (194, 154, 84)    # 棋盘背景色
LINE_COLOR = (30, 30, 30)    # 网格线颜色
STAR_POINT = (0, 0, 0)       # 星位点颜色
```

### 调整 AI 思考时间

在 `ai_move()` 方法中：

```python
# AI 思考延迟（毫秒）
think_time = 1500 if self.ai_difficulty == 2 else 800
# 可调整为：
# - 更快：1000 / 500
# - 更慢：2000 / 1000
```

## 📸 常见问题

### Q1: 为什么 AI 落子这么快？
A: AI 使用后台计算，几乎瞬间完成评估。如需增加思考时间，可以调整 `think_time` 参数。

### Q2: 如何查看保存的棋谱？
A: 在游戏目录下找到 `gomoku_*.txt` 文件，用记事本或任何文本编辑器打开即可。

### Q3: 悔棋能无限使用吗？
A: 是的，但只能撤销偶数步（玩家+AI 为一轮），且游戏未结束时才能悔棋。

### Q4: 自动保存的文件在哪里？
A: 与游戏程序在同一目录下，文件名格式为 `gomoku_YYYYMMDD_HHMMSS.txt`。

### Q5: 如何删除旧的棋谱文件？
A: 在游戏目录中使用文件管理器手动删除，或在 PowerShell 中执行：
```powershell
Remove-Item gomoku_*.txt
```
